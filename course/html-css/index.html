HTML là gì?

  • HTML(Hyper Text Markup Language): Là ngôn ngữ đánh dấu tiêu chuẩn để tạo ra trang web

  • HTML mô tả cấu trúc của một trang web

  • HTML bao gồm nhiều phần tử và HTML báo cho các trình duyệt biết cách hiển thị nội dung của các phần tử đó như thế nào
