Các thẻ HTML thông dụng

  • h1 - h6 : Là thẻ Heading để đánh dấu tiêu đề của văn bản
  • p : Là thẻ Paragraph để đánh dấu đoạn văn bản
  • a : Là thẻ anchor sử dụng như một đường link
  • img : Là thẻ đánh dấu hình ảnh
  • ul li : Là thẻ đánh dấu list danh sách
  • table : Là thẻ đánh dấu bảng
  • input : Là thẻ đánh dấu form nhập dữ liệu
  • button : Là thẻ đánh dấu nút nhấn
  • div : Là thẻ đánh dấu một khối giao diện
